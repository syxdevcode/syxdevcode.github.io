使用Naudio将Pcm转G711

我们在音频处理的时候经常会接触到PCM数据：它是模拟音频信号经模数转换（A/D变换）直接形成的二进制序列，该文件没有附加的文件头和文件结束标志。

声音本身是模拟信号，而计算机只能识别数字信号，要在计算机中处理声音，就需要将声音数字化，这个过程叫经模数转换（A/D变换）。最常见的方式是通过脉冲编码调制PCM(Pulse Code Modulation) 。
