Python3 中可以使用 json 模块来对 JSON 数据进行编解码，它包含了两个函数：

  • json.dumps(): 对数据进行编码。
  • json.loads(): 对数据进行解码。

json 模块主要提供了 dump、dumps、load、loads 方法对 JSON 数据进行编解码。
