Wow64

Wow64，全称是32bit Windows On 64bit Windows（64位Windows上的32位Windows）。

Windows系统的主要系统文件都是放在一个叫做System32的文件夹中的。为了能同时放下两套系统文件，Windows会在64位的系统上，增加了一个文件夹，叫SysWow64。
