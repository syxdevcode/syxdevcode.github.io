使防火墙生效

iptables 常用命令

在 target 里指定的特殊值：

  • ACCEPT – 允许防火墙接收数据包
  • DROP – 防火墙丢弃包
  • QUEUE – 防火墙将数据包移交到用户空间
  • RETURN – 防火墙停止执行当前链中的后续 Rules，并返回到调用链(the calling chain)中。
