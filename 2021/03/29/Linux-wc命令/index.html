Linux wc命令

发表于 2021-03-29 更新于 2022-09-27 分类于 Linux基础命令
本文字数： 526 阅读时长 ≈ 1 分钟

Linux wc 命令用于计算字数。利用 wc 指令可以计算文件的Byte数、字数、或是列数，若不指定文件名称、或是所给予的文件名为”-“，则 wc 指令会从标准输入设备读取数据。

语法 wc [-clw][--help][--version][文件...]

参数：

  • -c或–bytes或–chars 只显示Bytes数。
  • -l或–lines 显示行数。
  • -w或–words 只显示字数。
  • –help 在线帮助。
  • –version 显示版本信息。
