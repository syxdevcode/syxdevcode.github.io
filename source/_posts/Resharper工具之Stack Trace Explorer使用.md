# Resharper工具之Stack Trace Explorer使用

`Stack Trace Explorer` 显示剪贴板中的堆栈跟踪，并将其中的行转换为可单击的链接，可以直接导航到相关代码。

使用：

1，复制需要调试跟踪的代码，

2，打开Resharper-> Tools -> Browse Stack Trace
