## 算术运算符

| 运算符 | 描述 | 实例 |
| :-----:| :----: | :----: |
| + | 加 - 两个对象相加 | a + b |
| - | 减 - 得到负数，或是一个数减去另一个数 | a - b |
| * | 乘 - 两个数相乘，或是返回一个被重复若干次的字符串 | a * b |
| / | 除 - x 除以 y | a / b |
| % | 取模 - 返回除法的余数 | a % b |
| ** | 幂 - 返回x的y次幂 | a ** b |
| // | 取整除 - 向下取接近商的整数 | >>> 9//2  4  >>> -9//2  -5 |
