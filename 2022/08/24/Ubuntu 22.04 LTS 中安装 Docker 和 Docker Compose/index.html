Docker 依赖项

为了安装并配置 Docker ，你的系统必须满足下列最低要求：

  • 64 位 Linux 或 Windows 系统
  • 如果使用 Linux ，内核版本必须不低于 3.10
  • 能够使用 sudo 权限的用户
  • 在你系统 BIOS 上启用了 VT（虚拟化技术）支持 on your system BIOS（参考: 如何查看 CPU 支持 虚拟化技术（VT））
  • 你的系统应该联网
